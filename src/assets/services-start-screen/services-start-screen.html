Команда: Все сервисы

Интерфейс команды

Аналитика

Сервис для просмотра аналитики с подробным отображением данных по дням и построением графиков

Сервис расчётов

Сервис для расчетов грузового плана и/или остойчивости судна, подготовке документации для судохода или выхода из порта

Документация

Сервис для добавления, хранения и удобного доступа к документам судна

Сервис доп. расчетов

Сервис вспомогательных расчетов для определения остатка грузоподъемности судна и проверки драфт сюрвея

Настройки

Сервис по настройке продукта

Поддержка

Сервис для получения технической поддержки
